R코드 를 문서에 바로 반영해보자

```{r}

R code

```

의 형태로 적어주면 코드가 실행된다

plot(iris)를 안에 적어보자

```{r}

plot(iris)

```

plot(iris)

```{r, echo = FALSE}

plot(iris)

```

echo = FALSE 옵션의 경우 R 코드를 숨기고 결과문만을 보여준다

plot의 결과인 차트만 출력하고 싶을 때 사용하면 된다

```{r, eval = FALSE}

plot(iris)

```

eval = FALSE 옵션은 R 코드만을 보이고 계산을 하지 않는다

plot(iris)

문장을 쓰다가 중간에 R연산의 결과를 넣어야 할 경우도 있을 것이다.
